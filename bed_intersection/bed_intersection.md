#### Description

<p>Generate intersection position based on two bed file</p>

#### Usgae

```shell
usage: bed_intersection.py [-h] [-b1 BED1] [-b2 BED2] [-o OUT]

bed region intersection

optional arguments:
  -h, --help            show this help message and exit
  -b1 BED1, --bed1 BED1
                        bed1 path
  -b2 BED2, --bed2 BED2
                        bed2 path
  -o OUT, --out OUT     out path
```
